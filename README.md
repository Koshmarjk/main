# Консольный симулятор жизни

Простой консольный симулятор экосистемы «хищник–жертва».

## Структура проекта

```
life\\\_simulator/
├── main.py        # точка входа
├── ecosystem.py   # класс Ecosystem — управляет симуляцией
├── organism.py    # классы Organism, Herbivore, Predator
├── population.py  # класс Population — группа организмов
├── utils.py       # вспомогательные функции
└── README.md
```

## Классы

|Класс|Файл|Описание|
|-|-|-|
|`Organism`|organism.py|Базовый организм с энергией|
|`Herbivore`|organism.py|Травоядное, умеет пастись|
|`Predator`|organism.py|Хищник, умеет охотиться|
|`Population`|population.py|Группа одного вида|
|`Ecosystem`|ecosystem.py|Вся экосистема, цикл дней|

## Запуск

```bash
python main.py
```
